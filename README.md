# 图书馆管理系统 python1006

本项目是一个基于Python语言和Django框架开发的图书馆管理系统。

## 技术栈

- 后端：Python 3.x，Django
- 数据库：PostgreSQL
- 前端：HTML，CSS，JavaScript

## 功能模块

- 用户管理：注册、登录、权限控制
- 图书管理：图书信息录入、查询、借阅、归还
- 统计分析：借阅排行榜、图书分布图等

## 系统角色

- 系统管理员：负责用户管理和图书管理
- 普通用户：可以查询图书、借阅图书

## 运行环境

- 操作系统：兼容Linux和Windows
- Python版本：3.x
- 依赖管理：通过pip安装的第三方库

## 部署步骤

1. 安装Python环境
2. 使用pip安装Django和其他依赖
3. 配置数据库PostgreSQL
4. 运行项目迁移以创建数据库表
5. 启动Django开发服务器

- 基于成熟的Django框架，易于维护和扩展
